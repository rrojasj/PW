Venta de carros

Instrucciones: Crear una página web programada que tenga una demostración donde se pueda elegir un color diferente a un carro modelo de una agencia. El componente que el usuario va a interactuar para cambiar los colores puede ser como: radio, combo o carousel. Poner un banner en la parte superior de la página web abajo del encabezado.
